Добавляем кнопку на сайт

На сайтах, где много информации нужно ее как-то структурировать. Например, можно добавить кнопки, отсылающие на разные вкладки. Такие кнопки создаются с помощью тегов button, type, name эти теги помогут нам создать кнопку, а если нам нужно заблокировать какую-то кнопку, можно использовать тег disabled.

  • <!DOCTYPE html>
  • <html>
  • ㅤ<head>
  • ㅤㅤ<meta charset="utf-8">
  • ㅤㅤ<title>Название</title>
  • ㅤ</head>
  • ㅤ<body>
  • ㅤ<button type="button">Я кнопка</button>
  • ㅤ</body>
  • </html>
  • button - создаёт обычную кнопку.
  • type - определяет тип элемента ввода.
  • name - устанавливает имя элемента.
  • <!DOCTYPE html>
  • <html>
  • ㅤ<head>
  • ㅤㅤ<meta charset="utf-8">
  • ㅤㅤ<title>Название</title>
  • ㅤ</head>
  • ㅤ<body>
  • ㅤ<button type="button" disabled>Я заблокированная кнопка</button>
  • ㅤ</body>
  • </html>
  • disabled - блокирует доступ к кнопке.

Добавляем ссылку

На сайтах, где расписана какая-то информация нужно прикреплять ссылки с ее подтверждением. Скажем вы пишете сайт про некое животное, и что бы не расписывать, где живет это животное, чем питается и на сколько опасно, вы можете вкратце рассказать про него и прикрепить ссылки на сайты где всё подробно расписано.

  • <!DOCTYPE html>
  • <html>
  • ㅤ<head>
  • ㅤㅤ<meta charset="utf-8">
  • ㅤㅤ<title>Название</title>
  • ㅤ</head>
  • ㅤ<body>
  • ㅤㅤ<a href="ссылка">Указатель ссылки</a>
  • ㅤ</body>
  • </html>
  • a - используется для создания гиперссылок.
  • href - указывает URL-адрес, на который будет вести ссылка.

Какие ссылки есть?

  • https:// — протокол, по которому мы обращаемся к сайту.
  • site.ru/ — имя сервера.
  • /blog/index.html — путь к файлу.
