Создаем HTML страницу

Сайты в интернете, состоят из HTML – страниц, как книги. Книга – это сайт, а страницы — это ссылки. Давайте создадим свою первую HTML – страницу.

HTML-страницы создаются с помощью язык HTML, а код этого языка записывается в текстовых редакторах, например в обычном Блокноте от Windows, или же в сторонних программах. Введя в текстовый редактор, или же в сторонней программе код, написанный на языке HTML, а затем сохранив получившийся документ в файл с расширением .html мы получим HTML-файл, который можно просмотреть с помощью браузера.

  • <!DOCTYPE html>
  • <html>
  • ㅤ<head>
  • ㅤㅤ<meta charset="utf-8">
  • ㅤㅤ<title>Название</title>
  • ㅤ</head>
  • ㅤ<body>
  • ㅤ</body>
  • </html>
  • <!DOCTYPE html> - указывает на тип документа.
  • html - корневой тег документа, является контейнером веб-страницы.
  • head - в этом теге указываются основные данные, касающиеся генерации веб-страницы (заголовок, стили, скрипты, мета-теги и так далее).
  • meta - используется для указания метаданных страницы (в нашем случае тег указывает на кодировку страницы. Подробнее о мета-тегах читайте в статье Meta теги).
  • body - основной тег страницы. Представляет собой "тело" документа, внутрь которого помещаются остальные теги.
  • title - задает непосредственный заголовок страницы.

Затем сохраните получившийся документ, в виде файла с любым названием и расширением .html.
Теперь файл index.html, который вы сохранили, можно открыть в браузере и посмотреть на получившуюся HTML-страницу.
