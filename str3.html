Добавляем изображение на сайт

Сайты зачастую используют различные изображения, они своего рода являются простым объяснением вещей, животных или использования каких-то задач в жизни. Зачем подолгу расписывать как выглядит предмет если можно просто вставить изображение

  • <!DOCTYPE html>
  • <html>
  • ㅤ<head>
  • ㅤㅤ<meta charset="utf-8">
  • ㅤㅤ<title>Название</title>
  • ㅤ</head>
  • ㅤ<body>
  • ㅤ<img srl="Адрес изображения.jpg" alt="Изображение">
  • ㅤ</body>
  • </html>

При создании сайтов большинство разработчиков то же используют изображения. Вставить и отрегулировать изображение можно с помощью тегов по типу: img, srl, alt и т.д. все теги расписаны в низу, как и пример использования.

  • img - в HTML используется для вставки изображений на веб-страницы. Он поддерживает форматы GIF, JPEG, PNG, WebP, GIF, SVG.
  • srl - указывает путь к изображению.
  • alt - предоставляет текстовое описание изображения. Этот текст отображается, если изображение не может быть загружено.
  • width - задаёт ширину изображения.
  • height - задаёт высоту изображения.
  • src - указывает путь к изображению.
